# MediaForge

This project is a polished single-page web app for analyzing public media URLs and simulating premium MP3/MP4 download flows.

## Features
- Premium landing experience with animated cards and glassmorphism styling
- URL input, validation, drag-and-drop handling, and sample prompts
- Media analysis preview with audio and video format options
- Download queue with progress simulation and history tracking
- Local browser storage for recent download history

## Structure
- index.html: page structure and content sections
- styles.css: visual design, layout, responsive styling, and animations
- script.js: URL validation, sample analysis, queue simulation, history rendering, and toast notifications

## Run locally
Open index.html in a browser, or serve the folder with any static server such as:

```bash
python -m http.server 8000
```

Then open http://localhost:8000/.
